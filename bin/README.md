# E-Commerce Backend (Spring Boot)

This project is a simple e-commerce backend that I am building to understand how real backend systems work. The main goal is to learn how to handle common features like user login, product listing, cart management, and order processing using Java and Spring Boot.

##  What I am learning through this project

- How to structure a Spring Boot application using Controller, Service, and Repository layers  
- Applying basic and advanced Java concepts (OOP, interfaces, collections, streams, exceptions)  
- Building REST APIs and handling request/response flows  
- Using Spring Security and JWT for login and authentication  
- Working with MySQL and understanding how JPA/Hibernate handle entity relationships  
- Creating relationships like One-to-Many, Many-to-One, etc.
